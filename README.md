# Sign-Up Form Project

## Overview

This project is a recreation of a sign-up form page based on a provided
design file. The goal was to practice translating a static design into a
webpage using HTML and CSS.

## What I Learned

-   How to turn a design file into clean, structured HTML and CSS.
-   Planning and scaffolding page layout before coding.
-   Using flexbox for layout and spacing.
-   Adding a background image with a semi-transparent overlay for
    readability.
-   Styling form states using :focus and :invalid pseudo-classes.
-   Creating consistent UI elements like buttons, inputs, and headings.
-   Managing external assets such as fonts, images, and logos.

## Technologies Used

-   HTML5
-   CSS3
-   Git & GitHub
